Text Colors

Our text colors are designed for optimal readability and contrast.

Dark Text on Light Background

This is body text using our primary dark color (#192525) on a light background. This combination provides excellent readability with a contrast ratio of 16.1:1, well above the WCAG AA requirement of 4.5:1.

This is secondary text using our primary color (#334b4e) on a light background. This combination provides good readability with a contrast ratio of 6.8:1.

Light Text on Dark Background

This is body text using our primary light color (#cccfcf) on a dark background. This combination provides excellent readability with a contrast ratio of 9.2:1, well above the WCAG AA requirement of 4.5:1.

This is secondary text using our primary light color (#bac4c3) on a dark background. This combination provides good readability with a contrast ratio of 7.5:1.

White Text on Primary Background

This is body text using white on our primary color (#334b4e) background. This combination provides good readability with a contrast ratio of 6.8:1, well above the WCAG AA requirement of 4.5:1.

Dark Text on Light Primary Background

This is body text using our dark color (#192525) on our light primary color (#bac4c3) background. This combination provides excellent readability with a contrast ratio of 10.2:1.
